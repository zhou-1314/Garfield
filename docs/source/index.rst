Garfield: **G**raph-based Contrastive Le**ar**ning enables **F**ast S**i**ngle-C**el**l Embe**d**ding
========================================================================

Garfield is a novel graph representation algorithm with contrastive learning to embed cells into the same latent space. The joint embedding of cells allows Garfield to perform various types of single cell tasks, including scRNA-seq, scATAC-seq and scProtein clustering, batch correction, multi-omics integration and deconvolution of spatial data with scRNA-seq.
